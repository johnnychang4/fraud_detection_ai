## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Add your OpenAI API key to an `.env` file

   ```bash
   EXPO_PUBLIC_OPENAI_API_KEY=<your-openai-api-key>
   ```

3. Start the app

   ```bash
    npx expo start
   ```
